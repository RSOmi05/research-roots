

# Personal Academic Portfolio Website

## Overview
A clean, minimal, professional academic website with all the sections you described — built as a single-page app with smooth scrolling and modular, easily editable components.

## Pages & Sections

### Navigation
- Sticky top navbar with links: Home, About, Research, Publications, Projects, Teaching, CV, Contact
- Hamburger menu on mobile
- Active section highlighting on scroll
- Dark mode toggle in navbar

### Hero Section
- Large name heading, academic title, affiliation
- Circular professional headshot placeholder
- Short research summary
- CTA buttons: Download CV, View Publications, Contact Me
- Social icons row (Google Scholar, ResearchGate, LinkedIn, GitHub, Email)

### About Section
- Two-column layout (photo + bio) on desktop, stacked on mobile
- Research interests as styled bullet list
- Education timeline with degree, institution, year
- Skills grid (technical + research tools)

### Research Section
- Card grid layout with hover animations
- Each card: topic title, description, key contributions, link to papers

### Publications Section
- Filter tabs: Journal Articles, Conference Papers, Book Chapters, Working Papers
- Each entry: title, authors (your name bolded), journal/conference, year, DOI link, PDF button, copy BibTeX button
- Expandable abstract, sortable by year

### Projects Section
- Project cards with: title, role, duration, funding agency, description, outcomes, external link

### Teaching Section
- Course entries with: title, institution, semester/year, description
- Supervision subsection (thesis/undergrad/graduate)

### CV Section
- Embedded PDF viewer for CV preview
- Download CV button
- Quick summary highlights: Education, Experience, Awards, Grants

### Contact Section
- Email, office address display
- Contact form (name, email, message) with validation and success toast
- Optional embedded map placeholder

### Footer
- Copyright, quick nav links, social icons, "Last updated" date

## Design
- Minimal academic style with Inter font
- Color palette: deep navy primary, teal accent, white background
- Dark mode support
- Smooth scroll animations, subtle hover effects
- Mobile-first responsive design

## Technical Approach
- All content defined as editable constants in a single data file for easy replacement
- Each section as its own modular component
- SEO meta tags in index.html
- No backend needed — fully static, ready for deployment
